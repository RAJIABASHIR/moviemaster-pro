Description: MovieMaster Pro is a single-page movie catalog website where users can browse a curated list of movies in a clean, card-based layout. Each movie shows key details like title, poster, genre, and a short description, presented in a responsive grid that works nicely on both desktop and mobile screens. The UI focuses on simple navigation and a smooth viewing experience, making it ideal as a front-end portfolio project.

Technologies: React,React Router, Vite, JavaScript , HTML, Tailwind CSS, React-Toastify, responsive layout (Flexbox/Grid), Netlify deployment.
